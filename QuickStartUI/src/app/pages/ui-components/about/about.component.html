Welcome to MatechCoding QuickStart A startup project template for Quick Application Development. This application template provides a quick and convenient way to create ASP.NET Core 8.0 / Angular 17 applications. Technologies Template pages with Angular 17 and TypeScript RESTful API Backend using ASP.NET Core 8.0 Web API Database using Entity Framework Core Authentication/Authorization based on OIDC/OAuth2 and ASP.NET Core Identity API Documentation using Swagger Client-side built on Angular CLI Theming using Angular Material 16
